Tratar con archivos

Un sitio web consta de varios archivos:

  • contenido textual,
  • código,
  • hojas de estilo,
  • contenido audiovisual, etc.

Cuando estás construyendo un sitio web, necesitas ensamblar estos archivos en una estructura razonable y asegurarte de que pueden comunicarse entre sí. Tratar con archivos explica cómo configurar una estrutura de archivos razonable y qué temas tener en cuenta.

Conceptos básicos de HTML

Hypertext Markup Language (HTML) es el código que se utiliza para estructurar el contenido de tu web, y darle sentido y propósito. Por ejemplo,

  1. ¿el contenido de mi pagina web es un conjunto de párrafos, o una lista de viñetas?
  2. ¿Tengo imágenes insertadas en mi página? ¿Tengo una tabla de datos?

Sin abrumarte Conceptos básicos de HTML proporciona suficiente información para que te familiarices con HTML.
